# VoxelRenderer
A Minecraft like rendering engine made with OpenGL and OpenTK (C# binding for OpenGL)

## Controls
WASD - Move around<br>
Spacebar - Move up<br>
Left Shift - Move down<br>
Mouse - Look around<br>
F - Toggle wireframe mode<br>
ESC - Close window
