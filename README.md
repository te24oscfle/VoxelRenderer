# VoxelRenderer
A Minecraft like rendering engine made with OpenGL and OpenTK (C# binding for OpenGL)

## Controls
WASD - Move around
Spacebar - Move up
Left Shift - Move down
Mouse - Look around
F - Toggle wireframe mode
ESC - Close window
